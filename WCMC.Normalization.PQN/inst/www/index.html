Instruction

Probabilistic Quotient Normalization (Dieterle et al. 2006) assumes that biologically interesting concentration changes influence only parts of the NMR spectrum, while dilution effects will affect all metabolite signals. In case of urine spectra, dilution effects are caused, for example, by variations in fluid intake. Probabilistic Quotient Normalization (PQN) starts, with an integral normalization of each spectrum, followed by the calculation of a reference spectrum such as a median spectrum. Next, for each variable of interest the quotient of a given test spectrum and reference spectrum is calculated and the median of all quotients is estimated. Finally, all variables of the test spectrum are divided by the median quotient.

mainApp = function(input){
  library(pacman)
  pacman::p_load(data.table)
  input = gsub("\r","",input)
  cfile = strsplit(input,"\n")[[1]]
  df1 = as.data.frame(do.call(rbind,lapply(cfile,function(x){strsplit(x,"\t")[[1]]})),stringsAsFactors = F)
  colnames(df1) = as.matrix(df1[1,])[1,]
  df1 = df1[-1,]
  data = sapply(df1[,-1], as.numeric)

  reference = apply(data, 1, median)
  quotient = data / reference
  quotient.median = apply(quotient, 2, median)
  result = t(t(data) / quotient.median)


  result = cbind(data.frame(df1[,1]),result)
  rownames(result) = 1:nrow(result)
  colnames(result) = colnames(df1)
  fwrite(data.table(result),"PQN-normalization.csv")
  fwrite(data.table(result),"PQN-normalization.txt",sep = "\t")
  return(result)
}
